# Project Overview

## Ali Ahmed - Software Engineering Portfolio

This repository contains the source code for my professional software engineering portfolio. It is designed to be a high-performance, accessible, and visually striking representation of my work, methodologies, and technical capabilities.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Email Service**: EmailJS (Server-to-Server)
- **Deployment**: Vercel

## Core Objectives
1. **Performance**: Achieve near-perfect Lighthouse scores through server-side rendering, optimized assets, and efficient React component boundaries (`use client` only where strictly necessary).
2. **Aesthetics**: Implement a "Neo-Technical" design system that avoids generic tropes, utilizing noise textures, dynamic glassmorphism, and stark contrast to convey precision.
3. **Security**: Ensure zero secrets or configuration keys are exposed to the client browser by leveraging strict Server Actions.
4. **SEO & Discoverability**: Build a fully indexed site with dynamic `robots.ts` and `sitemap.ts` generation, officially verified via Google Search Console.

## Project Structure
- `/src/app`: Next.js 14 App Router pages, layouts, and server actions.
- `/src/components`: Reusable UI components and complex interactive elements.
- `/public`: Static assets, including SEO verification files.
- `/docs`: Comprehensive documentation detailing architectural decisions, UX evolution, and problem-solving methodologies.
